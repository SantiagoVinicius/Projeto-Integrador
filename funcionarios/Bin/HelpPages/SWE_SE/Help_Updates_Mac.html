Hjälp om startprogrammet för The Sims 3

Speluppdateringar

Ibland kommer The Sims 3 att erbjuda speluppdateringar som inför nytt innehåll eller löser kända problem.

Du kommer att få ett meddelande när det finns en uppdatering tillgänglig om du har Automatisk uppdatering markerat. Speluppdateringar kan även fås via panelen Speluppdateringar om du bestämmer dig för att inte uppdatera just då eller inte har markerat Automatisk uppdatering. Klicka på Uppdatera-knappen för att se om det finns några nya tillgängliga uppdateringar.

Vilken version av spelet som är installerat visas på menypanelen i startprogrammet.
