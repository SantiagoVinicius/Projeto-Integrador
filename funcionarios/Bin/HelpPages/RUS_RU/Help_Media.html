Данные

Снимки и видеоролики, сделанные в игре, отображаются на панели данных. Эти файлы должны быть упакованы в формат .Sims3Pack, после чего их можно будет опубликовать на веб-сайте игры. Кнопка Конвертировать упаковывает файлы и перемещает их на панель Загрузки. Конвертированные файлы с панели Загрузки будут отправлены на веб-сайт The Sims 3.

Материалы разбиты на несколько групп:

  • Видеоролики – видеоролики из игры, которые еще не конвертированы для публикации.
  • Снимки экрана - снимки экрана, которые еще не конвертированы для публикации.
  • Все – все материалы.

Пользователи могут выбрать один или несколько объектов. Кнопка Отменить выбор снимает имеющееся выделение.

Кнопка Удалить удаляет выбранные файлы из папки данных.

Выберите файл(ы) для отправки и щелкните на кнопке Конвертировать. Введите название и описание для каждого файла и выберите ОК. Готовые материалы появятся на панели Публикации. Вы получите сообщение, когда

Кнопка Показать файл поможет найти выбранный предмет на жестком диске и откроет нужную папку.
