Проверка и отмена подлинности

Проверка подлинности диска - это метод защиты от копирования, при котором для игры необходимо наличие диска The Sims 3. Этот метод используется при установке игры с диска. При каждом запуске игры проверяется подлинность диска The Sims 3.

Сетевая проверка подлинности производится однократно при каждом запуске игры. При этом методе проверки подлинности наличие диска с игрой в дисководе не требуется.

Если вы установили игру The Sims 3 посредством цифровой загрузки, сетевая проверка подлинности используется по умолчанию.

Отмена подлинности установки The Sims 3 позволяет запретить использование игры на отдельном компьютере при помощи сетевой проверки подлинности. Используя сетевую проверку подлинности, вы можете установить свою копию игры на 5 компьютерах. Если нужно будет запретить доступ к игре на одном из 5 компьютеров и установить игру на шестой, воспользуйтесь отменой подлинности установки The Sims 3. Если возникнут вопросы, обратитесь в службу технической поддержки.

Обновления

Иногда The Sims 3 будет предлагать обновить игру для загрузки материалов и исправления ошибок.

Если у вас отмечен пункт Автоматическое обновление, вам будут приходить сообщения о доступных обновлениях. Если пункт Автоматическое обновление не отмечен или вы хотите обновить игру позже, это можно сделать на панели Обновления. Щелкните на кнопку Обновить для проверки доступных обновлений.

Версия установленной игры отображается на главной панели запуска игры.
