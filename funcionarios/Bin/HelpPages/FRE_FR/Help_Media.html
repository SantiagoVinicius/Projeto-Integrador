Médias

Toutes les captures d'écran et vidéos venant du jeu sont affichées dans le panneau Médias. Ces fichiers doivent être rassemblés dans un fichier .Sims3Pack avant de pouvoir être partagés en ligne. Le bouton Convertir rassemble les fichiers de médias et déplace les fichiers convertis dans le panneau Chargements. Les vidéos converties et captures d'écran sont envoyées vers le site Web Les Sims 3 depuis le panneau Chargements.

Le contenu est trié en différents onglets :

  • Vidéo – Vidéo du jeu non convertie pour le chargement.
  • Captures d'écran - Les captures d'écran qui n'ont pas été converties pour le chargement.
  • Tout - Tout le contenu partagé.

Les utilisateurs peuvent sélectionner un ou plusieurs éléments. Le bouton Aucune sélection désélectionnera tous les éléments cochés.

Le bouton Supprimer supprime le ou les ensemble(s) de fichiers sélectionné(s) du dossier de médias de l'utilisateur.

Sélectionnez le ou les fichiers que vous souhaitez partager et cliquez sur le bouton Convertir. Entrez un nom et une description pour chaque fichier et cliquez sur Ok.
