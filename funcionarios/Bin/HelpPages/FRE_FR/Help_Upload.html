Chargements

Le contenu partagé en jouant au jeu Les Sims 3 sera affiché dans le panneau Chargements avant d'être partagé en ligne. Vous devez être connecté(e) avec un compte Les Sims 3 enregistré pour le jeu pour participer au partage de contenu sur le site Web Les Sims 3. Vous pouvez vous connecter ou créer un nouveau compte depuis le lien S'enregistrer sur la page Bienvenue.

Le contenu est trié en différents onglets :

  • SimsTerrains – Terrains avec Sims, terrains sans Sims, mondes.
  • Objets – Objets et style du mode Construction et du mode Achat.
  • Motifs – Motifs, matières de Créer un style.
  • Vidéo – Vidéos du jeu.
  • Captures d'écran – Captures d'écran du jeu.
  • Tout - Tout le contenu partagé.

Les utilisateurs peuvent sélectionner plusieurs éléments à charger sur le site Web Les Sims 3. Le bouton Aucune sélection désélectionnera tous les éléments cochés. Lorsque vous sélectionnez des captures d'écran ou vidéos, vous serez invité(e) à saisir un titre et une description. Les fichiers de médias sur votre disque dur gardent leur nom d'origine.

Le bouton Supprimer supprime le ou les ensembles de fichiers du dossier des chargements de l'utilisateur. Supprimer des fichiers avec le bouton Supprimer ne les supprime pas de l'échange Les Sims 3.

Le bouton Afficher le fichier localise le ou les éléments sélectionnés sur votre disque dur et ouvre le dossier.
