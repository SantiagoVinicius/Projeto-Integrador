Téléchargements

Le contenu téléchargé à partir de l'échange Les Sims 3 apparaît dans le panneau Téléchargements avec une image, un nom, un type de fichier et sa taille. Vous devez être connecté(e) avec un compte Les Sims 3 enregistré pour le jeu pour participer au partage de contenu sur le site Web Les Sims 3. Vous pouvez vous enregistrer ou créer un nouveau compte depuis le lien Se connecter sur la page Bienvenue.

Le contenu est trié en différents onglets :

  • SimsTerrains – Terrains avec Sims, terrains sans Sims, mondes.
  • Objets - Objets et styles du mode Construction et du mode Achat.
  • Motifs – Motifs, matières de Créer un style.
  • Tout - Tout le contenu partagé.

Les utilisateurs peuvent sélectionner un ou plusieurs éléments. Le bouton Aucune sélection désélectionnera tous les éléments cochés.

Le bouton Supprimer supprime le ou les ensemble(s) de fichiers du dossier des téléchargements de l'utilisateur. Le contenu installé ne sera pas supprimé du jeu.

Installation de contenu - Sélectionnez les éléments que vous souhaitez installer puis cliquez sur le bouton Installer. Un écran de chargement apparaitra au fur et à mesure que le ou les fichiers s'installent dans le client du jeu Les Sims 3. Cliquez sur OK lorsque c'est fini. Le contenu installé est indiqué par une icône dans la colonne Statut.

Le bouton Afficher le fichier localise le ou les objets sélectionnés sur votre disque dur et ouvre le dossier.
