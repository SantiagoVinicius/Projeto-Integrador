Authentification et annulation de l'autorisation

Autorisation avec disque est une méthode de protection sous copyright nécessitant que le disque Les Sims 3 soit dans le lecteur pour que vous puissiez y jouer. Si vous avez installé le jeu à partir d'un disque, il s'agit du paramètre par défaut. Lorsque le jeu est lancé, une vérification a lieu pour définir si le disque Les Sims 3 est un exemplaire valide.

L'Autorisation en ligne effectue une vérification en ligne unique lorsque le jeu est lancé. Cette méthode d'autorisation vous permet d'exécuter le jeu sans avoir mis le disque du jeu dans le lecteur.

Si vous avez installé Les Sims 3 à partir d'un Téléchargement numérique, l'Autorisation en ligne est la méthode de production sous copyright utilisée par défaut.

Annuler l'autorisation de cet ordinateur pour l'installation Les Sims 3 vous permet d'annuler l'autorisation d'un ordinateur spécifique utilisant l'Authentification en ligne. Avec l'authentification en ligne, vous pouvez autoriser jusqu'à 5 ordinateurs à exécuter le jeu. Si vous avez besoin d'enlever votre ordinateur de cette liste de 5 pour autoriser un autre ordinateur à exécuter le jeu, vous pouvez utiliser l'outil "Annuler l'autorisation de cet ordinateur pour l'installation Les Sims 3" pour le faire. Si vous avez des questions ou si vous avez besoin d'aide sur la procédure d'annulation des autorisations, contactez le service clients.

Mises à jour du jeu

De temps en temps, Les Sims 3 proposera des mises à jour du jeu pour apporter du nouveau contenu ou régler des problèmes connus.

Vous serez prévenu(e) lorsqu'une mise à jour sera disponible si vous avez coché Mises à jour automatiques. Les mises à jour du jeu sont également disponibles dans le panneau Mises à jour du jeu si vous décidez de ne pas mettre à jour tout de suite ou que vous n'avez pas coché Mises à jour automatiques. Cliquez sur le bouton Rafraîchir pour voir la disponibilité des mises à jour.

La version du jeu installée actuellement est indiquée dans le panneau principal du lanceur.
