Actualizaciones del Juego

Autentificación y Desautorización

La Autorización Desde el Disco es un método para protección del derechos de autor que requiere que el disco The Sims 3 esté en el lector para poder jugar el juego. Si instalaste el juego desde un disco, esta es la operación predeterminada. Cuando el juego es ejecutado se verifica que el disco The Sims 3 sea una copia válida.

La Autorización En Línea hará una revisión en línea por una sola vez cuando el juego sea ejecutado.Este método de autorización te permite correr el juego sin tener el disco dentro del lector.

Si instalaste The Sims 3 desde una Descarga Digital, la Autorización En Línea es el método para protección de derechos de autor utilizada.

Desautorizar Esta Máquina Para La Instalación The Sims 3 te permite desautorizar una máquina individual que utiliza Autentificación en Línea. Con Autentificación en Línea, puedes autorizar hasta 5 máquinas para que corran el juego. Si necesitas liberar tu máquina para que puedas autorizar otra para que corra el juego, puedes usar la herramienta "Desautorizar esta Máquina para The Sims 3" para hacerlo. Si tienes preguntas o necesitas ayuda con el proceso de desautorización, ponte en contacto con Servicio al Cliente.

Actualizaciones del Juego

Ocasionalmente, The Sims 3 ofrecerá actualizaciones para contenido nuevo o arreglar problemas conocidos.

Se te notificará cuando esté disponible una actualización si tienes seleccionado Actualizaciones Automáticas. Las actualizaciones de juego también están disponibles en el panel de Actualizaciones de Juego si decides no actualizar en ese momento o no seleccionaste Actualizaciones Automáticas. Haz Clic en el botón Refrescar para ver si hay actualizaciones disponibles.

La versión del juego instalada actualmente se muestra en el panel principal del launcher.
