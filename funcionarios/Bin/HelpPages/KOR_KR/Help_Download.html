다운로드

심즈 3 익스체인지에서 다운로드한 콘텐츠는 다운로드에 이미지, 이름, 유형, 파일 크기와 함께 표시될 것입니다. 심즈 3 웹사이트에서 콘텐츠를 공유하려면 게임을 등록하고 심즈 3 계정에 로그인해야 합니다. 처음 화면 페이지의 로그인 링크를 통해 로그인 하거나 심즈 3 계정을 새로 만드실 수 있습니다.

콘텐츠는 여러 항목으로 나뉘어 분류됩니다.

  • 심 - 심, 가족 구성원, 심 스타일 (의상, 액세서리, 머리모양).
  • 부지 - 심들이 있는 부지, 심들이 없는 부지, 세계
  • 사물 - 건축 및 구입 모드 아이템과 스타일
  • 패턴 - 패턴, 스타일 만들기 재료들.
  • 전체 - 모든 공유 컨텐트.

사용자는 하나 이상의 아이템을 선택할 수 있습니다. 선택 안 함 버튼은 모든 체크된 아이템의 체크를 해제합니다.

삭제 버튼은 사용자의 다운로드 폴더에서 선택된 파일 패키지(들)을 삭제합니다. 이것은 게임에 설치된 컨텐츠를 제거하지는 않습니다.

콘텐츠 설치 - 설치하고 싶은 아이템을 선택하여 설치 버튼을 클릭하십시오. 그러면 불러오기 화면이 뜨고 심즈 3 게임 클라이언트에게 파일(들)이 설치됩니다. 완료되면 확인을 누르십시오. 설치된 콘텐츠는 상태 컬럼에 아이콘으로 표시됩니다.

파일 표시 버튼은 귀하의 하드 드라이브 상에서 선택한 아이템의 위치를 찾아 폴더를 엽니다.
