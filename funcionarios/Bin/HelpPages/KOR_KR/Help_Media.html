미디어

게임 진행 중 발생한 모든 스크린샷은 미디어 패널에 표시될 것입니다. 이 파일들은 온라인 상으로 공유하기 전에 Sims3Pack 파일로 압축되어야 합니다. 변환 버튼은 미디어 파일을 패지키화 하여 변환된 파일을 업로드 패널로 이동시킵니다. 변환된 동영상 및 스크린샷은 업로드 패널로부터 심즈 3 웹사이트로 업로드 됩니다.

콘텐츠는 여러 항목으로 나뉘어 분류됩니다.

  • 동영상 - 업로드를 위해 변환되지 않은 게임 동영상.
  • 스크린샷 – 업로드를 위해 변환되지 않은 게임 스크린샷.
  • 전체 - 모든 공유 컨텐트.

사용자는 하나 이상의 아이템을 선택할 수 있습니다. 선택 안 함 버튼은 모든 체크된 아이템의 체크를 해제합니다.

삭제 버튼은 사용자의 미디어 폴더에서 선택된 파일 패키지(들)을 삭제합니다.

공유하고 싶은 파일(들)을 선택하고 변환 버튼을 누르십시오. 각 파일에 대한 이름과 설명을 입력하고 확인을 클릭합니다. 올바른 포맷으로 변환이 완료되면 업로드 패널에 미디어가 나타납니다.

파일 표시 버튼은 귀하의 하드 드라이브 상에서 선택한 아이템의 위치를 찾아 폴더를 엽니다.
