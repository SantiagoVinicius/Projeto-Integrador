Installed Content

The Installed Content panel tracks and manages custom content installed in your game.

Content is sorted into different tabs:

  • Sims – Sims, households, Sim styles (clothing, accessories and hair).
  • Lots – Lots with Sims, lots without Sims, worlds.
  • Objects – Build and Buy Mode objects and styles.
  • Patterns – Patterns, Create a Style materials.
  • All - All content

Users can select one or more items to manage. The Select None button will deselect all checked items.

The following actions are available:

  • Run Without Custom Content – The game will run in a ‘safe mode’ by temporarily disabling all installed custom content.
  • Uninstall – Uninstalls and removes selected content from the game client. Uninstall will not remove the original file(s) downloaded from The Exchange. Those files are deleted from the Downloads panel.
