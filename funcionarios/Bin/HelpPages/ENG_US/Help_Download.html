Downloads

Content downloaded from The Sims 3 exchange will appear in the Downloads panel along with an image, name, type and the size of the file. You must be logged in with a game-registered The Sims 3 account to participate in content sharing on The Sims 3 website. You can log in or create a new account from the Log In link on the Welcome page.

Content is sorted into different tabs:

  • Sims – Sims, households, Sim styles (clothing, accessories and hair).
  • Lots – Lots with Sims, lots without Sims, worlds.
  • Objects – Build and Buy Mode objects and styles.
  • Patterns – Patterns, Create a Style materials.
  • All – All shared content.

Users can select one or more items. The Select None button will deselect all checked items.

The Delete button removes the selected file package(s) from the user’s download folder. It will not remove installed content from the game.

Installing Content - Select the items you wish to install and click the Install button. This will bring up a loading screen as the file(s) are installed into The Sims 3 game client. Click OK when complete. Installed content is noted with an icon in the Status column.

The Show File button will locate selected item(s) on your hard drive and open the folder.
