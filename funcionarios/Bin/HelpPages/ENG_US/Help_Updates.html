Game Updates

Authentication and De-Authorization

Disc-Based Authorization is a copyright protection method that requires The Sims 3 disc to be in the drive in order to play the game. If you installed the game from a disc this is the default behavior. When the game is launched The Sims 3 disc is verified to be a valid copy.

Online Authorization will do a one-time online check when the game is launched. This method of authorization allows you to run the game without having the game disc in the drive.

If you installed The Sims 3 from a Digital Download, Online Authorization is the default copyright protection method used.

De-Authorize This Machine For The Sims 3 Installation allows you to de-authorize an individual machine that uses Online Authentication. With Online Authentication, you can authorize up to 5 machines to run the game. If you need to free up your machine so that you can authorize another machine to run the game, you can use the "De-authorize This Machine for The Sims 3" tool to do so. If you have questions or need help with the de-authorization process, contact Customer Support.

Game Updates

Occasionally The Sims 3 will offer game updates to introduce new content or to fix known issues.

You will be notified when an update is available if you have Automatic Updates checked. Game updates are also available on the Game Updates panel if you decide not to update at that time or do not have Automatic Updates checked. Click the Refresh button to see if any new updates have been made available.

The current installed game version is displayed on the main launcher panel.
