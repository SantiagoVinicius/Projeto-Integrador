Media

All screenshots and video taken from the game will be displayed in the Media panel. These files must be packaged into a .Sims3Pack file before they can be shared online. The Convert button packages media files and moves converted files to the Uploads panel. Converted Video and Screenshots are uploaded to The Sims 3 website from the Uploads panel.

Content is sorted into different tabs:

  • Video – Game video that has not been converted for upload.
  • Screenshots – Screenshots that have not been converted for upload.
  • All – All shared content.

Users can select one or more items. The Select None button will deselect all checked items.

The Delete button removes the selected file package(s) from the user’s media folder.
