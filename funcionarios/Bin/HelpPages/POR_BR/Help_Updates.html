Autenticação e Desautorização

A Autorização por Disco é um método de proteção de direito autoral que exige que o disco do The Sims 3 esteja na unidade para você jogar. Se você tiver instalado o jogo com um disco, esse será o padrão de procedimento. Quando o jogo é executado, o disco do The Sims 3 é verificado como uma cópia válida.

A Autorização On-line realiza uma única verificação quando o jogo é executado. Esse método de autorização permite que você execute o jogo sem ter o disco de jogo na unidade.

Se você tiver instalado The Sims 3 a partir de um Download Digital, a Autorização On-line é o método de proteção de direito autoral a ser usado.

Desautorizar esta Máquina para Instalação do The Sims 3 permite que você desautorize uma máquina individual que usa a Autenticação On-line. Com a Autenticação On-line, você pode autorizar até 5 computadores a executar o jogo. Se você precisar liberar sua máquina para pode autorizar outra, pode utilizar a ferramenta "Desautorizar esta Máquina para Instalação do The Sims 3" a fim de fazer isso. Se tiver dúvidas ou precisar de ajuda no processo de desautorização, entre em contato com o Suporte ao Cliente.

Atualizações de Jogo

Ocasionalmente, o jogo The Sims 3 oferecerá atualizações para introduzir conteúdo novo ou reparar problemas.

Você será notificado quando houver uma atualização disponível, se a opção Atualizações Automáticas estiver ativada. As atualizações também ficam disponíveis no painel Atualizações de Jogo se você decidir não atualizar na hora ou não tiver a opção Atualizações Automáticas ativada. Clique no botão Atualizar para conferir se há novas atualizações disponíveis.

A versão do jogo instalada atualmente é exibida no painel principal do inicializador.
