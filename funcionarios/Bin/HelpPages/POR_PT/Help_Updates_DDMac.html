Autenticação e Desautorização

Autorização com Base em Disco é uma protecção de direitos de autor que requer que o disco de Os Sims™ 3 esteja na unidade para que possas jogar. Se instalaste o jogo a partir de um disco, este é o comportamento predefinido. Quando o jogo é iniciado, o disco de Os Sims™ 3 é verificado para determinar se é um exemplar válido.

Autorização Online fará uma única verificação online quando o jogo é executado. Este método de autorização permite-te executar o jogo sem teres o disco do jogo na unidade.

Se instalaste Os Sims™ 3 através de Download Digital, a Autorização Online é o método predefinido de protecção de direitos de autor.

Desautorizar esta Máquina para Instalação de Os Sims™ 3 permite-te desautorizar uma máquina individual que utilize a Autenticação Online. Com a Autenticação Online podes autorizar até 5 máquinas para que executarem o jogo. Se precisares de libertar a tua máquina para que possas autorizar outra máquina para executar o jogo, podes utilizar a ferramenta "Desautorizar esta Máquina para Os Sims™ 3" para o fazer. Caso queiras fazer perguntas ou precises de ajuda sobre o processo de desautorização, contacta a Assistência ao Cliente.

Actualizações para o Jogo

Ocasionalmente Os Sims™ 3 oferecerá actualizações para o jogo para apresentar novos conteúdos ou resolver problemas conhecidos.

Se marcares a caixa Actualizações Automáticas, receberás uma notificação quando estiver disponível uma actualização. As actualizações para o jogo também estão disponíveis no painel Actualizações para o Jogo se escolheres não actualizar nesse momento ou Actualizações Automáticas não estiver marcado. Clica no botão Actualizar para verificares se estão disponíveis novas actualizações.

A versão do jogo actualmente instalado é apresentada no painel principal do programa de lançamento.
