Ajuda para o Programa de Lançamento do Jogo Os Sims™ 3

Actualizações para o Jogo

Ocasionalmente Os Sims™ 3 oferecerá actualizações para o jogo para apresentar novos conteúdos ou resolver problemas conhecidos.

Se marcares a caixa Actualizações Automáticas, receberás uma notificação quando estiver disponível uma actualização. As actualizações para o jogo também estão disponíveis no painel Actualizações para o Jogo se escolheres não actualizar nesse momento ou Actualizações Automáticas não estiver marcado. Clica no botão Actualizar para verificares se estão disponíveis novas actualizações.

A versão do jogo actualmente instalado é apresentada no painel principal do programa de lançamento.
